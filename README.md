# Contact Manager

A simple contact management application built using React. User can add, delete, update and search contacts here.

## Features
- Add new contacts with name and email id
- Edit existing contacts
- Delete contacts
- Search contacts with real-time filtering


## Tech Stack used

- React with Hooks
- React Router
- Tailwind CSS
- Axions for api calls
- Heroicons

## Installation

1. clone the repository
2. open the project
3. install dependencies: npm install 
4. start development server: npm run dev
